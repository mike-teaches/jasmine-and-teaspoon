## Ruby on Rails Application

This project was generated using:

``` 
  $ rails new rails_server
```

## Setup

### Install dependencies

```
  $ bundle install
```

### Run the server

```
  $ bundle exec rails s
```

### Open your browser

* [http://localhost:3000](http://localhost:3000)
